# 🚀 Projects

> Applied projects that bring the lecture concepts together into working neural network implementations.

## Philosophy

Each project in this directory builds on the concepts from the lectures, combining them into complete, runnable applications. Projects start simple and grow in complexity as we learn more techniques.

## Projects

| Project | Description | Concepts Used | Status |
|---------|-------------|---------------|--------|
| [Spiral Classifier](./spiral_classifier/) | Classify 2D spiral data using a manually-built dense network | Layer_Dense, forward pass, spiral data | ✅ Complete |

### Planned Projects

- **Moon Classifier** — Binary classification on the sklearn `make_moons` dataset
- **Digit Classifier** — MNIST handwritten digit recognition from scratch
- **Moons & Spirals Comparison** — Compare how different architectures handle different data shapes
- **Feature Visualization** — Visualize what each layer of the network learns

## How to Run

```bash
# Run the spiral classifier
python projects/spiral_classifier/spiral_classifier.py

# Or from the project directory
cd projects/spiral_classifier
python spiral_classifier.py
```

## Results

Each project directory contains a `results/` folder where output plots, logs, and visualizations are stored. Check the specific project's README for details.
